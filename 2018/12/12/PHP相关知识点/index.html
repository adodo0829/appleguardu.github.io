关于php的一些知识点

发表于 2018-12-12 | 分类于 PHP

主要是总结一些网页中php的使用方式…

php基础

基本语法

一般一个语言的语法无非是由变量,数据类型,运算,流程控制,函数,让就是封装的一些方法,库等…php可以与html一起混合使用,其语法与js大致相同,这里便不做太多介绍了,以下有几个注意点

注意点:
    1. 文件创建: filename.php 代码块: <?php 代码块 ?>

    2. 变量声明: 要以$开头, 区分大小写

    3. 关于字符串: 
        a. ""双引可以解析出变量名,''单引号不行;
        b. 字符串的拼接语法使用点 . 来拼接

    4. 关于输出:
        a. 简单数据类型(字符串,数值等)输出: echo 
        b. 复杂数据类型(数组,对象等)输出:  print_r() 或者 var_dump()输出详细信息

    5. 关于数组
        a. 索引数组: 与js一样, $arr = [1,2,3,4]
            使用for循环遍历, 长度用count($arr)方法获取

        b. 关联数组: 键值对形式的数组, $arr = ['key' => 'value',...];
            b1. 取值使用$arr['key']方式
            b2. 遍历使用foreach方法
                foreach($arr as $key => $value){
                    //循环体里$key就是当前遍历到的元素的$key
                    //循环体里$value就是当前遍历到的元素的$value
                }

    6.指令式的混编写法
         <?php
            $array = ['php', 'java', 'python', 'ruby'];
            //循环开始
            for($i = 0; $i < count($array); $i++):
        ?>
        <!-- 每遍历一个元素就创建一个li标签 -->
        <li><?php echo $array[$i]; ?></li>
        <!-- 循环结束 -->
        <?php endfor; ?>

    7.文件引入:  include "文件路径.php";

表单交互

html中的表单(form), 主要是用来向后台提交数据…

  • form表单,页面提交数据的入口

    <!-- action属性是提交的地址，代表提交到哪个服务器页面, 默认get请求方式 -->
<form action="./result.php">
// 必须要加name属性!!!!!, 后台通过name属性获取传递的数据
<input type="text" name="userName" placeholder="请输入用户名">
<input type="submit" value="提交">
</form>

  • 提取表单数据

    上面的表单数据会被提交到result.php文件,这时我们需要提取数据值
通过$_GET['上面表单的name属性值']即可获取到: 如$res = $_GET['userName'];

isset可以用来判断是否存在某个值 isset($res);返回true/false

  • get请求和post请求

    1. get请求:
<form action="login.php" method="GET">
    <input type="text" name='id' placeholder="请输入账号">
    <input type="password" name='pwd' placeholder="请输入密码">
    <input type="submit" value='登录'>
</form>
// 提交数据后的处理页面的链接为: 127.0.0.1/login/login.php?id=hello&pwd=123
// get请求会将我们输入的数据: 
        a. 先将我们输入的数据进行url的拼接
        b. 然后以明文的方式发送服务器处理页面; 处理页面通过$_GET['name属性']方式取值进行处理
// 所以get请求的特点:
        1. 简单,传输快,主要用于获取
        2. 不安全
        3. 长度有限制(2KB)
        4. 浏览器缓存
// 注意点:
    a标签的跳转，也可以理解为是一种get请求。因为get请求本身就是通过网址的跳转来进行的。
    所以,我们可以直接构造href属性去请求其他页面, 传入参数通过$_GET方式获取

2. post请求:
<form action="login.php" method="POST">
    <input type="text" name='id' placeholder="请输入账号">
    <input type="password" name='pwd' placeholder="请输入密码">
    <input type="submit" value='登录'>
</form>
// 提交数据后的处理页面的链接为: 127.0.0.1/login/login.php
// post请求会将我们输入的数据: 
        a. 先将我们输入的数据放入到请求体中
        b. 然后发送服务器处理页面; 处理页面通过$_POST['name属性']方式取值进行处理
// 所以post请求的特点:
        1. 主要用于提交数据
        2. 长度无限制
        3. 相对安全(抓包可以抓到post的提交数据)

  • 文件上传

    // 通过表单标签<input type="file">来上传

1.上传到处理页面 
    <!-- 表单里仅仅一个文件上传元素，提交到file.php -->
        <form action="file.php" method='POST' enctype='multipart/form-data'>
            // 需要指定enctype属性
            <input type="file" name='icon'>
            <input type="submit">
        </form>

2.接受,处理文件: 使用$_FILES['name属性']接受, 返回结果为关联数组

<?php
// 处理上传文件
// 1.接受文件
$files = $_FILES['files']; 
var_dump($files);
// 返回一个关联数组, 有5个属性:
//  1.name：保存的是文件名
// ​ 2.type：保存的是文件类型
// ​ 3.tmp_name：保存的是客户端上传来的文件临时保存的路径
// ​ 4.error：错误信息代码
// ​ 5.size：文件大小

// 2.保存文件到本地:  拿到临时路径, 指定文件新的路径地址
$oldPath = $files['tmp_name'];
$newPath = "./data/".$files['name'];

// 3.转移到本地: 使用move_uploaded_file(old, new)方法
$res = move_uploaded_file($oldPath, $newPath);

if ($res):
    echo "success";
    echo $res;
else:
    echo "failed";
endif;
?>

3.文件读写操作
    a. file_put_contents('文件路径',写入内容);
    b. file_get_contents('文件路径')

4.文件存在判断: 返回布尔值
    file_exists('文件路径');

数据库交互

  • 增删改查

    1. 增删改
    // php操作数据库
    // 1. 建立链接: 服务器地址, 主机用户名, 密码, 数据库名
    $link = mysqli_connect('127.0.0.1', 'root', 'hh0127', 'userInfo');

    // 2.准备sql语句
    // 增
    $sql = "insert into user(userName, description) values('胡华', '你最帅啦')";
    // 删
    $delete = "delete from user where id = 2";
    // 改
    $sql = "update user set userName='huhua',description='我是哈哈' where id=5";

    // 3.执行语句: mysqli_query(数据库链接, sql语句)  返回值: 是否成功
    $res = mysqli_query($link, $sql);
    $affect_row = mysqli_affected_rows($link);  // 返回受影响的行数

    // 4.关闭链接
    mysqli_close($link);

2. 查询

    // 1.建立连接
    $link = mysqli_connect('localhost', 'root', 'hh0127', 'userInfo');

    // 2.sql查询语句
    // $sql = "select * from user";
    // 查询语句
    $sql = "select userName,description,id from user";
    $res = mysqli_query($link, $sql); // 返回的是一个对象,数据表结构

    // 3.取数据: 使用,mysqli_fetch_all(执行sql语句后的对象)  
    // 传入参数MYSQLI_ASSOC或者1, 可以将返回的数组内的 键 改为 字段名

    // $result = mysqli_fetch_all($res, MYSQLI_ASSOC); 
    $result = mysqli_fetch_all($res, 1); // 返回一个二维数组: 他的元素为: 一个个关系型数组
    echo $result[2]['description'];

    // 复杂类型输出
    print_r($result);

    // 断开连接
    mysqli_close($link);
